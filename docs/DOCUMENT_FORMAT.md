# Формат документа SOPR

Новые документы сохраняются как UTF-8 JSON с рекомендуемым расширением
`.sopr.json`. Изображение больше не является контейнером состояния и
экспортируется отдельной командой меню.

Минимальная структура:

```json
{
  "format": "sopr-scheme-document",
  "version": 1,
  "task_type": "beams",
  "canvas": {"width": 400, "height": 250},
  "common": {},
  "settings": {},
  "text": "",
  "task": {
    "version": 1,
    "payload": {}
  },
  "extensions": {}
}
```

- `format` и `version` задают общий формат файла.
- `task_type` — стабильный ID из реестра типов, а не локализованный заголовок.
- `canvas`, `common`, `settings` и `text` содержат состояние интерфейса.
- `task.version` версионирует модель конкретного типа независимо от envelope.
- `task.payload` содержит модель задачи.
- `extensions.section` сохраняет редактор поперечного сечения там, где он есть.

## Модель и безопасность

Payload поддерживает JSON-значения и несколько внутренних маркеров:

- `$type: object` — экземпляр модельного класса из allowlist выбранного типа;
- `$type: tuple` — кортеж;
- `$type: ndarray` — числовой NumPy-массив;
- `$ref` — ссылка на другой объект внутри того же payload.

Имена Python-модулей в файл не записываются и из файла не импортируются.
Каждый stable task ID имеет явный список допустимых классов и отдельную таблицу
миграций. Неизвестная версия, класс, ссылка, настройка или тип значения
отклоняются до переключения активного документа. Размер входного файла
ограничен 10 MiB, а размер холста — 10000×10000.

Запись выполняется через временный файл в том же каталоге с последующим
атомарным `replace`.

## Работа в приложении

В меню File операции разделены:

- **Открыть документ…** — читает только JSON-документ;
- **Сохранить документ…** — записывает `.sopr.json`;
- **Экспортировать изображение…** — создаёт PNG/JPEG без скрытого sidecar;
- **Импортировать старый доверенный .dat…** — отдельный legacy-import.

Старые `.dat` используют Python pickle и могут выполнять код. Приложение не
ищет их автоматически по хешу изображения. Импорт доступен только после явного
выбора команды, файла и подтверждения предупреждения. После импорта документ
следует сохранить в новом формате.
